This folder contains the main release of the core corpus REDEWIEDERGABE.

Folder "tsv" contains the data in column-based plain text format.
Folder "xmi" contains the data in XMI-format, as used by the annoation framework DKPro (https://dkpro.github.io/). NOTE: We provide XMI files as a convenience for DKPro users, but offer no further support for this format.
Folder "xml" contains the data in TEI-complient XML format.

For more detailed information please consult the documentation.
