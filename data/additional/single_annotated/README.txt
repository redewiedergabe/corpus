This folder contains additional material (complete texts and text samples) annotated with the full annotation system but only by single annotators.

Folder "full_fict" contains complete fictional texts. (NOTE: The annotation guidelines regarding the speaker annotation have been adjusted for some of these texts.)
Folder "full_nonfict" contains complete non-fictional texts.
Folder "samples" contains fictional and non-fictional text samples (about 500 tokens for fictional, and 200-500 tokens for non-fictional)

For more detailed information please consult the documentation.
