This folder contains a RELAX-NG schema that adapts the TEI Module for Linguistic corpora to the annotation schema of project REDEWIEDERGABE and its documentation. It relates to the xml files in the planet subfolders.

"tei_rw_corpus_schema.rng" is the RELAX-NG schema.
"tei_rw_corpus.xml" is the TEI customization file.
"tei_rw_corpus_doc.html" is the documentation of the XML schema.

For more detailed information please consult the documentation.
