This folder contains all core corpus samples annotated by the human annotator with the pseudonym "mercury" in TEI compliant XML format.
Each file contains the text of one sample, its metadata and the annotations of project REDEWIEDERGABE.

The corresponding RELAX-NG syntax schema that adapts the TEI Module for Linguistic corpora to the annotation schema of project REDEWIEDERGABE and its documentation can be found in data/additional/primary/resources_xml.

For more detailed information please consult the documentation.
